# APPLE-Motor-Control-
AIM

To calculate the required motor speed using the distance–time relationship for robotic motion control.

PROCEDURE
Initialize the input values: distance and time
Apply the formula: speed = distance / time
Substitute the given values
Compute the motor speed
Display the result
CODE
# Input values
distance = 20   # in meters
time = 8        # in seconds

# Speed calculation
speed = distance / time

# Output
print("Motor Speed:", speed, "m/s")
OUTPUT

Motor Speed: 2.5 m/s

RESULT

The motor speed is calculated as 2.5 m/s, which ensures the robot can travel the given distance within the specified time.
